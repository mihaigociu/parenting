## Essay Overview

The essay is organized in six parts:

1. **What a Child Is** — the philosophical starting point: the child as a relational being, not an individual project
2. **The Philosophy in Three Concepts** — amae (甘え), sunao (素直), ganbaru (頑張る)
3. **The Social Architecture** — wa (和), shudan seikatsu, uchi/soto (内/外)
4. **The Daily Practices** — soine, soji, mimamoru, kosodate
5. **What to Borrow — and What Not To** — honest cross-cultural translation
6. **The Shadow** — kyōiku mama, maternal pressure, romanticization critique
